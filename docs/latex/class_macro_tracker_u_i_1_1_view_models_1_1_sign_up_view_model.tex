bool? Macro\+Tracker\+UI.\+View\+Models.\+Sign\+Up\+View\+Model.\+Is\+Sign\+Up\+Valid (\begin{DoxyParamCaption}\item[{out string}]{prompt\+Message}{}\end{DoxyParamCaption})}



Checks if the sign-\/up is valid. This includes\+: 


\begin{DoxyItemize}
\item Checking if the username is null or empty.
\item Checking if the username already exists.
\item Checking if the password is strong.
\item Checking if the passwords match. If everything is valid, adds the user to the database and assigns a successful prompt message. If the username is null or empty, does nothing and the prompt message is null. Otherwise, assigns an error prompt message.
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em prompt\+Message} & The prompt message to be assigned based on the validation result.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true\+: if the sign-\/up is valid with the prompt message\+: "{}\+Sign up successfully"{}. false\+: if the sign-\/up is invalid with one of these prompt messages\+: "{}\+Username has already existed!"{}, "{}\+Password is not strong!"{}, "{}\+Passwords do not match!"{}. null\+: no action is taken and the prompt message is not assigned. 
\end{DoxyReturn}
