◆ IsSignUpValid()

bool? MacroTrackerUI.ViewModels.SignUpViewModel.IsSignUpValid ( out string promptMessage)

Checks if the sign-up is valid. This includes:

  • Checking if the username is null or empty.
  • Checking if the username already exists.
  • Checking if the password is strong.
  • Checking if the passwords match. If everything is valid, adds the user to the database and assigns a successful prompt message. If the username is null or empty, does nothing and the prompt message is null. Otherwise, assigns an error prompt message.
Parameters
promptMessageThe prompt message to be assigned based on the validation result.
Returns
true: if the sign-up is valid with the prompt message: "Sign up successfully". false: if the sign-up is invalid with one of these prompt messages: "Username has already existed!", "Password is not strong!", "Passwords do not match!". null: no action is taken and the prompt message is not assigned.
